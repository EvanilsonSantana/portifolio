Auxiliador Home-Office

O projeto foi criado sobre uma proposta de atividade para disciplina de sistemas embarcados da CESAR School.

O usuário poderá via internet, trancar a porta do escritório e avisar para as pessoas lado de fora que está em reunião. O objetivo do projeto foi auxiliar a comunicação das pessoas que trabalham em home-office, seja em um quarto ou escritório. O projeto propõe facilitar a comunicação de quem ta dentro do escritorio/quarto com quem ta fora.

Para mais detalhes sobre o projeto, documentação, tecnologias utilizadas e outros, confira o link do repositório: hackster.io

Rede Neural Siamesa Usando One-Shot learning para Reconhecimento Facial

Este projeto foi desenvolvido como parte da disciplina "Tópicos Avançados em Inteligência Artificial" na CESAR School.

Nossa proposta consistiu na implementação de redes neurais siamesas utilizando one-shot learning para reconhecimento facial. Este desafio nos levou a criar um google colab e elaborar um short paper.
